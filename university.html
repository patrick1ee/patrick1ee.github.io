University Projects

Group and invidual projects which I have worked on as part of my degree programme from 2020 to present

ROP Shellcode Compiler For Antivirus Evasion

A project done with a partner in which we built a tool that take certain Shellcodes and compile them into a ROPchain exploit, given binary vulnerable to buffer overflow. The exploit can then be used to hijack control flow and execute code fragments already within the compiled binary executed arbitrarily to achieve the same result as executing the shellcode and bypassing defences such as executable space protection, code signing and ASLR.

This project gave huge insight into security considerations in software development as well as its low level underpinnings

Profitability Analysis of Trading Agents

An indvidual project in which I ran simulations, involving different trading agents in a wide range of market conditions and then ran testing to compare performance. Amongst the agents analysed were various zero-intelligence algorithms, simplified stategies such as Giveaway and Shaver and more advanced strategies such as the Stochastic Hillclimber and Parameterized Response models.

This project was invaluable in learning the underpinnings of successful trading algorithms as well as rigorous statistical analysis

AES Key Recovery and Countermeasures

An individual project undertaken during the latter term of my third year, aiming to explore both power analysis attacks on AES-128 implementations and respective countermeasures. This centered around a micro-controller with firmware to perform encryption and decryption operations, interacting with a host via UART. I developed this firmware and demonstrated an efficient key recovery attack against it. I then implemented counter-measures for this attack, using operation permutation and masking, and wrote a specification considering further threats and counter-measures for other functionalilty required for TLS communication

Druid Drum Circle (Video Game Project)

A team project in which we developed an immersive rhythm game in which players play drums in time to descending "beat" markers (think Guitar Hero) to cast spells which would cause procedural generation of trees and foliage, themed to each level (mountains, forest, beach). Challenges included building drum hardware, analysis of music tracks, procedural tree growth and integrating other hardware (Kinect, multi-screen immersive tent).

This was a good exposure to an intense team project using AGILE and SCRUM methodology.
